# 📖 project
프로젝트 / 공모전 정리 


본 레포지토리는 데이터 분석, 머신러닝·딥러닝, AI 서비스 기획, 공공 데이터 활용을 중심으로 수행한 프로젝트들을 정리한 **portfolio repository**입니다.
