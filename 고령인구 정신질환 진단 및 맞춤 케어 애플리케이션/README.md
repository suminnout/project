# 고령인구 정신질환 진단 및 맞춤형 케어 AI 애플리케이션

본 프로젝트는 초고령사회 진입에 따라 심화되는 고령인구 정신질환 문제를 해결하기 위해 공공 빅데이터와 AI 기반 멀티모달 분석을 활용한 정신질환 진단 및 맞춤형 케어 서비스를 설계·제안한 프로젝트입니다.

단순한 헬스케어 앱이 아닌, **진단 → 의료·복지 연계 → 지속적 케어**까지 이어지는 공공 데이터 기반 정신건강 관리 시스템을 목표로 함

## 🎯 문제 인식 (Why)
### 고령인구 정신건강 관리의 구조적 한계
- 고령사회 진입에 따른 **우울증·불안장애·불면증 유병률 증가**
- 진단 방식이 설문·대면 상담 중심 → **주관성 및 접근성 한계**
- 독거노인 증가, 지방 거주 고령층의 **의료 접근성 문제**
- 진단 이후 치료·복지·생활 관리로의 **연계 부족**

즉, 정신질환 데이터와 서비스는 존재하지만 **조기 진단과 지속 관리가 하나의 흐름으로 연결되지 못하는 구조적 문제**가 존재

---

## 🧠 해결 아이디어
### 공공 데이터 기반 멀티모달 정신질환 진단 및 케어 시스템
- 설문 데이터 + 음성 + 영상 데이터를 결합한 **객관적 진단**
- 질환 유형 및 중증도 분류를 통한 **맞춤형 의료·복지 연계**
- 단발성 진단이 아닌 **지속 가능한 정신건강 관리 서비스** 제공

---

## 🛠 서비스 구현 기술
### 핵심 기술 구성

#### 1️⃣ 멀티모달 데이터 분석
- 설문 지표: PHQ-9(우울), GAD-7(불안), ISI-K(불면)
- 음성 데이터: MFCC 기반 발화 특징 추출
- 영상 데이터: 얼굴 키포인트 기반 표정·행동 분석

#### 2️⃣ AI 기반 진단 모델
- 음성 분석: **DenseNet**
- 영상 분석: **PoseNet**
- CNN / U-Net 기반 질환 및 중증도 분류 모델
- 설문·음성·영상 데이터 융합 분석 구조 설계

#### 3️⃣ 공공 데이터 연계
- 건강보험심사평가원 병·의원 및 약국 데이터 활용
- 사용자 위치 기반 의료기관 추천
- 질환 및 중증도 기반 자치단체 복지 정책 매칭
