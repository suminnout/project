# 👨‍👩‍👧‍👦 유니스크 UNISK : 디지털 시대의 사회적 소외 계층을 위한 키오스크 


본 프로젝트는 디지털 전환 과정에서 키오스크 사용에 어려움을 겪는 고령자·장애인 등 **디지털 소외 계층의 접근성 문제를 해결하기 위해 설계된 데이터 기반 음성 인식 키오스크 서비스**입니다.

단순 UI 개선이 아닌, **데이터 리터러시 관점에서 문제를 정의하고 음성 인식(STT)과 사용자 행동 데이터를 활용한 서비스 구조**를 제안한 프로젝트

## 🏆 프로젝트 성과
- **2023 강원대학교 LINC 3.0 사업단 캡스톤 디자인 데이터 리터러시 아이디어 경진대회 최우수상**

---

## 🔍 프로젝트 개요
- 프로젝트명: **UNISK (Universal + Kiosk)**
- 프로젝트 성격: LINC 3.0 캡스톤 디자인 · 데이터 리터러시 아이디어 경진대회
- 대상 도메인: 디지털 소외 계층(고령자·장애인) 키오스크 접근성
- 핵심 키워드: 데이터 리터러시 · 음성 인식(STT) · 유니버설 디자인 · 포용 기술

---

## 🎯 문제 인식 (Why)
### 디지털 전환 속 사회적 소외 문제
- 키오스크 확산으로 **대면 서비스 감소**
- 고령자·장애인의 디지털 기기 사용 부담 증가
- 시간 제한·복잡한 UI로 인한 **심리적 위축**
- 디지털 역량 부족이 곧 **사회적 배제**로 이어지는 구조

즉, 기술은 발전했지만 **모든 사용자를 고려하지 않은 디지털 환경이 새로운 장벽을 생성**하고 있음

### 음성 인식 기반 유니버설 키오스크 (UNISK)
- 화면 조작이 아닌 **음성 중심 인터랙션**
- 사용자 행동·음성 데이터를 활용한 **맞춤형 메뉴 추천**
- 디지털 역량에 관계없이 누구나 사용할 수 있는 키오스크 환경 제공

---

## 🛠 서비스 구현 아이디어
### 핵심 구성 요소

#### 1️⃣ 음성 인식(STT) 기반 주문
- 음성 명령으로 메뉴 탐색 및 주문
- 소음 환경에서도 높은 인식률을 고려한 STT 모델 활용
- 반복 주문·선호 메뉴 데이터 축적

#### 2️⃣ 사용자 맞춤형 추천
- 주문 이력 및 음성 데이터 기반 메뉴 추천
- 고령자·초보 사용자에게 단순화된 선택 흐름 제공

#### 3️⃣ 유니버설 디자인
- 연령·장애 유무와 관계없이 사용 가능한 인터페이스
- 시각 정보 최소화, 음성 피드백 중심 설계

---

## 🔄 서비스 워크플로우
1. 사용자 음성 입력  
2. STT 기반 명령 인식  
3. 사용자 유형·이력 기반 메뉴 추천  
4. 음성 안내를 통한 주문 진행  
5. 결제 및 주문 완료  

---

## 🖥 서비스 구성 예시
- 음성 안내 중심 키오스크 UI
- 복잡한 메뉴 구조 제거
- 선택 단계 최소화

---

## 🚀 기대 효과
### 사회적 효과
- 디지털 소외 계층의 **자립적 서비스 이용 환경 조성**
- 키오스크 사용에 대한 심리적 장벽 완화

### 기술적 효과
- 음성 데이터 기반 사용자 행동 분석 가능성 제시
- 포용적 기술(Accessible Tech) 적용 사례 제시

### 공공·교육적 효과
- 데이터 리터러시 기반 사회 문제 해결 사례
- 공공 서비스 및 민간 키오스크 확장 가능성

## 🌍 확장 가능성

## 🧑‍💻 프로젝트를 통해 배운 점
- 데이터 리터러시는 **분석 기술이 아니라 문제를 바라보는 관점**
- 기술 도입보다 **사용자 맥락 이해가 서비스 완성도를 결정**
- 사회 문제 해결형 프로젝트에서 기획과 구조 설계의 중요성
