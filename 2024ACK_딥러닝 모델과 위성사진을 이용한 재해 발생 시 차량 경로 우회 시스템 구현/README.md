# 딥러닝 모델과 위성사진을 이용한 재해 발생 시 차량 경로 우회 시스템

본 연구는 자연·인적 재해 발생 시 **위성사진과 딥러닝 기반 건물·도로 인식을 활용하여  
위험 지역을 회피하는 차량 경로 우회 시스템을 구현**한 연구입니다.

---

## 📄 논문 정보

- **논문명**: 딥러닝 모델과 위성사진을 이용한 재해 발생 시 차량 경로 우회 시스템 구현  
- **학술대회**: ACK 2024 학술발표대회 논문집 (31권 2호)  
- **주요 키워드**: 위성영상 · U-Net · 재해 대응 · 경로 탐색  
- **RISS 링크**: https://www.riss.kr/link?id=A109574499

---

## 🔍 연구 개요

- 위성사진을 입력으로 **도로 및 건물 영역을 U-Net 기반 Semantic Segmentation으로 검출**
- 재해 전·후 건물 마스크 비교를 통해 **건물 손상률 산출**
- 손상 건물 인접 도로를 제외한 영역에서 **A\* 알고리즘 기반 최단 우회 경로 탐색**
- 재난 상황에서 **안전이 확보된 이동 경로 제공**을 목표로 함

---

## 🧠 사용 기술

- **Deep Learning**: U-Net (Semantic Segmentation)
- **Data**: DeepGlobe, xView2 위성 이미지 데이터셋
- **Path Planning**: A\* Algorithm
- **Framework**: Python 기반 딥러닝 환경

---

## 📌 연구 의의

- 재해 상황에서 **정량적 건물 손상 정보 + 경로 탐색을 결합**
- 기존 수작업·현장 조사 중심 재난 대응 방식의 한계 보완
- 피난, 구호 물자 수송 등 **실제 재난 관리 시나리오 적용 가능성 제시**
