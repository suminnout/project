# 💊 Medicycle: 시민 참여형 폐의약품 안전수거 플랫폼

본 프로젝트는 시민 참여를 기반으로 한 **폐의약품 안전 수거 및 관리 플랫폼 Medicycle**을 제안, 데이터 분석과 입지 선정 모델을 통해 **수거함 설치 효율과 시민 참여율을 동시에 개선**하는 것을 목표로 함

---

## 🏆 프로젝트 성과
- **2025 춘천시 데이터 활용 해커톤 은상 수상**
- 주관: 춘천시, 강원도민일보
- 데이터 분석 기반 정책 인사이트 도출 및 정책 제안의 실효성과 분석 완성도를 인정받음

---

- **프로젝트명**: Medicycle (메디사이클)
- **프로젝트 성격**: 데이터 기반 정책·플랫폼 기획
- **대상 지역**: 춘천시
- **핵심 키워드**: 폐의약품 · 시민 참여 · 입지 선정 · PCA · MCLP

---

## 🎯 문제 인식

- 폐의약품의 부적절한 폐기로 인한 **환경·수질 오염 문제**
- 수거함 설치는 이루어졌으나 **시민 인지도 및 참여율 저조**
- 설치 이후의 **운영·성과 관리 부재**

---

## 🧠 해결 방안

### 시민 참여형 폐의약품 수거 플랫폼

- 시민 대상 **폐의약품 수거 위치 안내 및 참여 유도**
- 수거 이력·포인트 기반 참여 보상 구조
- 행정 담당자를 위한 **수거 현황 모니터링 대시보드**

---

## 📊 데이터 기반 입지 선정 모델

### 1️⃣ PCA 기반 수요지수 산출
- 25개 행정·환경·인구 관련 지표 활용
- 주요 지표:
  - 유동성 지수
  - 환경 민감 지수
  - 의료 접근 지수
  - 인구 취약 지수
- PCA를 통해 **종합 수요지수 도출**

### 2️⃣ MCLP 기반 최적 입지 선정
- 수요지수 + 제약 조건을 결합한 **최적 수거함 위치 도출**
- 대규모 주거지, 의료기관 인근 우선 고려
